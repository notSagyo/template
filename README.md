# **Website Template**
Template to quickly initialize my personal web projects.
## Compile code
Compile CSS with:

```
gulp buildCss

gulp watchCss
```
For SCSS files:
```
gulp buildSass

gulp watchSass
```
Add ``Dist`` at the end to minify and/or add other dist processors to the pipeline:
```
gulp buildCssDist

gulp buildSassDist
```
